# Strategy Comparison Case Study

## Overview

Comparing strategies is useful because strong standalone results do not always
mean a strategy is the best fit. A side-by-side view helps answer practical
questions:

* which strategy delivered the strongest risk-adjusted return
* which one was more directional or more selective
* whether a custom strategy is meaningfully better than a simple baseline

This case study uses the current comparison workflow and the
deterministic artifacts generated by it.

## Strategies Compared

The intended comparison set for this example is:

* `momentum_v1`
* `mean_reversion_v1`
* `buy_and_hold_v1`

## Comparative Insights

`buy_and_hold_v1` is the current leader in the validated three-strategy
single-run leaderboard. In the present `features_daily` dataset, passive long
exposure compounds far more strongly than the two active rule-based strategies,
even though it carries very large drawdown.

`mean_reversion_v1` finishes second by Sharpe ratio. It still loses money in
absolute terms over this run, but it declines much less severely than
`momentum_v1`, which is enough to give it the stronger risk-adjusted score of
the two active strategies.

`momentum_v1` is the weakest performer in the validated run. It stays exposed
most of the time, but the current dataset regime produces a near-total loss and
the lowest Sharpe ratio in the comparison set.

## Key Takeaways

The main takeaway is that the comparison workflow is now operating
end to end for all three example strategies, and the resulting leaderboard is
fully reproducible.

The current dataset favors the simple `buy_and_hold_v1` baseline over both
active strategies. That makes the baseline an important benchmark rather than a
mere placeholder.

The active strategies remain useful because they show distinct behavior profiles
under the same artifact and comparison pipeline, but the present run does not
support a claim that either active strategy beats passive exposure on this
dataset snapshot.
